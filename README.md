# TRIAD PERFORMANCE

Strategic performance coaching for the Spinningfields professional—precision coaching, metabolic optimization, and postural restoration at 18 Leftbank.

## View the site

Open `index.html` in a browser, or use Cursor’s **Simple Browser** (`Cmd+Shift+P` → “Simple Browser: Show” → enter the full path to `index.html`).

## Stack

- Single-page HTML with [Tailwind CSS](https://tailwindcss.com) (CDN)
- [Lucide](https://lucide.dev) icons
- Google Fonts: Playfair Display (headings), Inter (body)
